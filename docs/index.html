Intro

json-summary was created to simplify the display of complex data to users.

The module attempts to extract important structural information from a JSON object in order to only show users the most important information relevant to their development. Among its benefits:
  • Removes repetitive information in datasets (e.g. long arrays)
  • Smaller representation is better for interface and large datasets on a server

Usage

Adding to your project

in browser (unpkg)

<link rel="stylesheet" href="https://unpkg.com/json-summary@0.2.0/summarizer.css"> <script src="https://unpkg.com/json-summary@0.2.0/dist/json-summary.min.js">

in node (npm install json-summary)

const jsonSummary = require("json-summary");

Creation:

let jsonSum = new jsonSummary();

Creation (with default options):

let jsonSum = new jsonSummary({ indentation: " ", indentCount: 2, showExampleValue: true, startExpanded: false, arraySampleCount: 10 });
OptionTypeDescription
indentation String the character used for indentation whitespace
identCount Number the number of whitespace characters to indent with per nested level
showExampleValue Boolean Whether or not the example values should be displayed for basic types
startExpanded Boolean whether the sections of the object should begin open or closed
arraySampleCount Number the number of elements to sample and summarize for arrays

Summarizing:

let data = { a: [2, 8, 24], b: {x: "y"} }; let summary = jsonSum.summarize(data); let output = jsonSum.printSummary(summary);
