Intro

json-summary was created to simplify the display of complex data to users.

The module attempts to extract important structural information from a JSON object in order to only show users the most important information relevant to their development. Among its benefits:
  • Removes repetitive information in datasets (e.g. long arrays)
  • Smaller representation is better for interface and large datasets on a server

Usage

Adding to your project:

<link rel="stylesheet" href="https://unpkg.com/json-summary@0.1.3/summarizer.css"> <script src="https://unpkg.com/json-summary@0.1.3/dist/json-summary.min.js">

Creation:

let jsonSum = new jsonSummary();

Creation (with default options):

let jsonSum = new jsonSummary({ indentation: " ", indentCount: 2, showExampleValue: true, startExpanded: false });

Summarizing:

let data = { a: [2], b: {x: "y"} }; let summary = jsonSum.summarize(data); let output = jsonSum.printSummary(summary);
